# You54f Homebrew tap

Personal formulae for the [Homebrew](https://brew.sh) package manager.

```bash
brew tap you54f/brew
```

## Cirrus CLI

To install a fork of [Cirrus CLI](https://github.com/you54f/cirrus-cli) run:

```bash
brew install you54f/brew/saffus
```
or
```bash
brew install saffus
```

## Passh

To install [Passh](https://github.com/clarkwang/passh) run:

```bash
brew install you54f/brew/passh
```
or
```bash
brew install passh
```

## krunvm / buildah / libkrun / libkrunvm
